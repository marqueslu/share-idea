Dicas para criação do seu projeto:

Descrição: Um resumo breve e convidativo sobre seu projeto.

Foto: Todo o projeto deve ter uma imagem de capa que expresse seu verdadeiro propósito.

Conteudo: Deve conter toda a história e informação sobre o projeto. Tente utilizar o máximo possível de imagens e vídeos para que seu projeto seja o mais atrativo.

Criar Projeto: Após seu projeto ser finalizado, basta clicar no botão ‘Criar Projeto’ para que seu projeto seja criado. Para visualizar ou editar clique no menu ‘Meus Projetos’.

Salvar Rascunho: Você também pode salvar o seu projeto sem publicá-lo, basta clicar no botão ‘Salvar Rascunho’, assim vai poder dar continuidade ao seu projeto em outro momento.
